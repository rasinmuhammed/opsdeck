# Support And Compatibility

## Supported direction

OpsDeck is optimized for:

- FastAPI
- SQLAlchemy 2.x async engines
- Pydantic v2
- Python 3.10+

## Support expectations

- New features should land with tests
- Docs examples should match the shipped API
- Security-sensitive behavior should prefer explicit configuration over hidden defaults

## Release expectations

- CI should pass before release
- Changelog entries should describe user-facing changes
- Compatibility-impacting changes should be called out clearly

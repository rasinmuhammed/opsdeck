# Release Plan: 1.1.0

## Release positioning

OpsDeck 1.1.0 is the release where the project becomes clearly FastAPI-native, AI-agent-friendly, and visually differentiated.

## Core message

OpsDeck is the FastAPI-native admin for async SQLAlchemy teams.

## What to highlight

- developer-obsessed Matrix UI refresh
- `ModelAdmin` for serious admin configuration
- model-level permissions and row scoping
- bulk actions and relationship search
- explicit audit model support
- stronger docs for humans and coding assistants

## Release checklist

- ensure docs build and GitHub Pages deploy from `main`
- create GitHub release `v1.1.0`
- publish PyPI release from GitHub release workflow
- unarchive or refresh PyPI project status if needed
- include screenshots or a short GIF in the release notes
- announce docs pages for AI assistants, recipes, comparison, and FAQ

## Recommended announcement angles

### Short version

OpsDeck 1.1.0 is out: a sharper Matrix UI, a stronger FastAPI-native admin API, and better docs for both developers and coding assistants.

### Developer version

If you build with FastAPI and async SQLAlchemy, OpsDeck 1.1.0 gives you explicit registration, `ModelAdmin`, row scoping, permissions, bulk actions, audit hooks, and a much stronger docs and UX story.

### AI-era version

OpsDeck 1.1.0 is built for the age of AI coding assistants: clearer docs, LLM-readable project files, predictable integration points, and copyable recipes for real FastAPI apps.
